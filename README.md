# Kalléssia Leme · Psicóloga de Mulheres

Site institucional disruptivo e interativo para a psicóloga **Kalléssia Leme** (CRP 09/15380).

> "Amor-próprio não é luxo, é sobrevivência."

## ✨ Recursos
- **Visual em degradê** (azul → violeta → magenta → rosa) com glassmorphism e cards com brilho.
- **Spotlight do mouse** ("escuro do mouse") revelando o conteúdo conforme o cursor.
- **Cards 3D interativos** com tilt e glow que seguem o ponteiro.
- **Vídeos 4K** (Pexels, uso livre) de fundo no hero, seção de análise e agendamento.
- **Análise gratuita** — quiz de 5 passos que gera uma recomendação personalizada de serviço.
- **Agendamento** com calendário + horários que monta a mensagem e envia direto ao **WhatsApp** da doutora com nome, data, horário, modalidade e assunto pré-preenchidos.
- **Som ambiente** opcional (WebAudio) acionado pela logo da **Kairos Digital** (marca d'água).
- 100% responsivo, sem dependências/build — HTML + CSS + JS puro.

## 🗂 Estrutura
```
index.html      · marcação e seções
styles.css      · design system, degradê, glow, spotlight
script.js       · interatividade, quiz, calendário, WhatsApp
assets/img/     · retratos, logo da marca, ícones, logo Kairos
assets/video/   · clipes 4K (Pexels)
vercel.json     · headers de cache
```

## 🚀 Deploy
Site estático — deploy automático na Vercel a cada push no GitHub.
